About Clear Recipes

What is Clear Recipes?

Clear Recipes is a simple recipe viewer that strips away the clutter and presents recipes step-by-step. No more scrolling through life stories, ads, and pop-ups. Just the ingredients and instructions you need, one step at a time.

Features

  • Step-by-step recipe viewing
  • Ingredients shown per step
  • Automatic timer detection
  • Dark mode support
  • Works on 100+ recipe websites
  • Save and share your favorite recipes
  • Swipe navigation on mobile

Bookmarklet

Add a bookmarklet to your browser to instantly send any recipe page to Clear Recipes with one click.

To create the bookmarklet:

  1. Drag this link to your bookmarks bar: Clear Recipe
  2. Or, create a new bookmark manually with this URL:
javascript:(function(){window.location='https://recipes.rybergs.com/?url='+encodeURIComponent(window.location.href)})();

On iPad/iPhone: Create a bookmark of any page, then edit it and replace the URL with the JavaScript code above. When you're on a recipe page, tap the bookmarklet to load it in Clear Recipes.

Chrome Extension

A Chrome extension is also available for one-click recipe loading. Install it in developer mode:

  1. Download the extension from the GitHub repository
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode" (toggle in top right)
  4. Click "Load unpacked" and select the chrome-extension folder
